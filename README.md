Decision Tree Implementation using ID-3


Requirments:


	Above Python 3, Tested on Python 3.6
	Pandas Library, You can install by typing “pip install pandas” on cmd


Working:


	It first takes dataset in the form of “csv”


	Calculates entropy of whole dataset


	Calculates IG of each feature and selects with max IG.


	Then recursively build tree based on data.


Results:


	Result is a nested dictionary that is used to display tree in Understandable format.


Inspirations:


	Code is not in Object Oriented Fashion you can convert it into classes.
